# The Edge of Silence (TEoS) beta 1.0

Атмосферная игра-лабиринт: игрок ориентируется по звуку, ряби и коротким вспышкам видимости.

## Быстрый старт

1. Создать и активировать виртуальное окружение:
   - Windows PowerShell: `.\venv\Scripts\Activate.ps1`
2. Установить зависимости:
   - `pip install -r requirements.txt`
3. Запустить игру:
   - `python main.py`

## Управление

- `WASD` / `Стрелки` — движение
- `Shift` — бег
- `ЛКМ (зажать)` — заряд броска камня
- `ЛКМ (отпустить)` — бросок
- `E` — поднять камень
- `Esc` — пауза
- `F3` — debug overlay
- `F5` — перезагрузка шейдеров
- `F6` — смена режима уровня (debug)

## Особенности реализации

- Рендер: гибрид `ModernGL` (вода) + `pygame` (мир/UI)
- Шейдеры вынесены в `shaders/`
- Настройки и рекорды сохраняются в `%APPDATA%\TheEdgeOfSilence\`:
  - `config\settings.json`
  - `data\teos.db`
- Базовые автотесты находятся в `tests/`

## Сборка EXE

- Актуальная инструкция по сборке: `BUILD_EXE.md`
- Для запуска на другом ПК передавайте всю папку `dist\TheEdgeOfSilence\`, а не только `exe`
